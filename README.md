# git-issue-viewer

![license](https://img.shields.io/badge/license-MIT-blue.svg)

A Simple Git Issues Viewer made with React and fluxible

## Installation

```bash
$ npm install
$ npm run dev
```
Now hit `http://localhost:3000` on your browser.

## Run Unit Tests

```bash
$ npm test
```
